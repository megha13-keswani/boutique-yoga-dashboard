# Boutique Yoga Studio Dashboard

A responsive dashboard for managing yoga studio classes, members, and bookings.

## Features

- Responsive design
- Class schedule
- Member search
- Booking form
- Light/Dark theme
- Form validation
- Loading and empty states

## Tech Stack

- HTML5
- CSS3
- JavaScript
